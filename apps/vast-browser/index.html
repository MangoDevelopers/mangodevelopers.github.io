Why VastBrowser?

Built for the remote

An on-screen cursor and scroll mode designed for D-pad navigation — browse any website from your couch, no mouse needed.

Real browser engine

Powered by Mozilla's browser components. The Pro flavor bundles the full GeckoView engine for consistent rendering on any device.

Ad blocking (Pro)

The Pro flavor supports content-blocking extensions like uBlock Origin for a cleaner, faster web on your TV.

Free & automatic updates

Every release is free, published openly on GitHub, and built automatically — grab the latest version anytime.

How to install

  1. Download the APK for your device (on your phone or computer).
  2. Transfer it to your TV — the easiest way is the free Send Files to TV app on both devices, or use a USB drive.
  3. On the TV, enable Install unknown apps for your file manager in Settings.
  4. Open the APK on the TV and install. Developers can also run adb install VastBrowser-arm64-v8a.apk.
